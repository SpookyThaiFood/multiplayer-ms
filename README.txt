# Welcome to online-multiplayer minesweeper using p5.js

This is a version of Daniel Shifman's Bee Sweeper that has been modified to be played together remotely.



# Usage

Use node to launch server.js and connect to "localhost:3000", or whatever domain you specify.

OR

Play this at: https://multiplayer-beesweeper.glitch.me/



# Key bindings

'f' : Flags the tile underneath the mouse
